# Sentinel-Dispatch

Motor académico de despacho para unidades de emergencia médica en la Región de Coquimbo.

El proyecto combina triaje basado en un subconjunto MPDS, ruteo A* sobre grafo vial OSM y una función de costo multiobjetivo para priorizar asignaciones de ambulancias.

> Proyecto académico. No corresponde a un sistema certificado ni debe usarse para operación real de salud.
